## 🧠 Philosophy

> *"Don't just simulate — build something that moves in the real world."*

I believe the best way to learn engineering is to build things that actually work — robots that navigate real rooms, code that runs on real hardware, and systems that fail to teach you why. Every bug is a lesson. Every sensor glitch is a puzzle.
